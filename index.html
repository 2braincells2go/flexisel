Flexisel

As long as there has been jQuery there have been image carousels created with jQuery. The famous jCarousel (http://sorgalla.com/jcarousel/) has been out since 2006, the same year the first version of jQuery was released. However, with the explosion of users now using their mobile devices for their internet browsing — on iPhones, iPads, and the like — it becomes necessary for jQuery plugins to support a good user experience on these devices. Enter Flexisel, the responsive image carousel with options specifically available for adapting the carousel for mobile and tablet devices.
